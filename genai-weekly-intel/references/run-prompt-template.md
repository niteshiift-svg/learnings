# Run Prompt Templates

Copy-paste prompts to trigger the genai-weekly-intel skill.

---

## Full Weekly Brief

```
GenAI weekly brief for the week of [Month Day] to [Month Day, Year].
```

```
What happened in GenAI this week? Full brief.
```

---

## Single-Player Snapshot

```
Single-player update for [Company] this week.
```

```
What's new with [OpenAI / Anthropic / Google / AWS / Microsoft / xAI] this week?
```

---

## With Specific Date Range

```
GenAI weekly brief for February 24 to March 2, 2026.
```

---

## With LinkedIn Post Output

```
GenAI weekly brief this week, and also give me a LinkedIn post based on the top insight.
```

---

## With Deck Output

```
GenAI weekly brief this week, and format a version I can use in a slide deck.
```

---

## Save to File

Add to any prompt above:
```
...and save it to my notes.
```

---

## Abbott-Specific Filter

```
GenAI weekly brief this week. Flag anything relevant to enterprise knowledge management, healthcare AI, or regulatory AI.
```
